## Keyword Usage Best Practices

### ✅ Do:
- Use keywords naturally in sentences
- Include in H1, H2, title, meta description
- Use variations and semantic alternatives
- Focus on user intent, not just keyword stuffing
- Use long-tail keywords in blog posts

### ❌ Don't:
- Repeat exact keyword more than 2-3 times in H1/H2
- Force keywords into awkward sentences
- Ignore semantic variations
- Optimize for keywords with no search volume
- Ignore user experience for keyword density
